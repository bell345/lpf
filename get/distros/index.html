Linux Distributions

A Linux distribution is a set of applications bundled with a Linux kernel (the core operating system stuff) to "complete" the package, such as desktop environments (where the windows and icons live), tools for all kinds of purposes (file management, system settings) and terminal emulators (where the fancy text and commands live).

  • Ubuntu

    Ubuntu is the most popular distribution of Linux, and is used by millions worldwide. It features the Unity environment, with a side-bar application launcher.

  • Linux Mint

    Linux Mint is a popular alternative, based on Ubuntu. It features a choice of default environments and a variety of tools suitable for the power user.

  • Arch Linux

    Arch Linux is not for the average Linux user. It features a bare-bones installation that the user can configure to his or her preferences. Often a distribution of choice for lightweight systems or for experienced Linux gurus.
