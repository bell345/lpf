What is Linux?

Linux, put simply, is an open-source operating environment. It performs the same function for a computer as Windows or Mac OS X, except that it is free to use, distribute and modify in any way imaginable.

What sets it apart is not only its price; Linux is highly customisable, and people can choose from any number of distributions, which are different implementations of the underlying Linux operating system that contain different programs, features and levels of support.

For example, Ubuntu is a highly popular version of Linux that millions of people use everyday. It is considered to be the most reliable and easy to use Linux distribution due to its intuitive user interface, assistive applications, support and similarities to many other GUI operating systems, such as Windows.

On the other hand, Arch Linux is a barebones installation of Linux, with only a few essentials which provide a foundation for lots of other programs that the user can install. It is considered to be a complex version of Linux because it doesn't hold your hand and can easily lead to a personalised system with only the programs and features that the user requires.

As you can see, each distribution of Linux can be a world away from another, and there are hundreds to choose from, such as Debian, Mint, Gentoo and Red Hat. Linux offers a unique computing experience that is geared towards the user's preferences.
